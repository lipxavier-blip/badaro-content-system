# badaro-content-system

## Visão Geral
Este projeto é um sistema de gestão e produção de conteúdo.

## Instruções para o Claude
- Siga as convenções de tom definidas em `skills/tone-checker.md`
- Consulte `memory.md` para contexto acumulado do projeto
- Use `skills/learn.md` para registrar aprendizados relevantes

## Estrutura do Projeto
```
badaro-content-system/
├── CLAUDE.md               # Instruções para o Claude
├── memory.md               # Memória persistente do projeto
└── skills/
    ├── tone-checker.md     # Verificação de tom e voz
    └── learn.md            # Registro de aprendizados
```
